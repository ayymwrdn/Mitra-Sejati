Website ini berfungsi sebagai sarana untuk mengenalkan perusahaan kepada audiens, baik itu calon klien, mitra bisnis, maupun publik secara umum. Dengan tampilan yang profesional, informatif, dan mudah diakses. Dengan tampilan yang profesional, informatif, dan mudah diakses, website company profile dapat menjadi media yang efektif untuk menampilkan informasi penting tentang perusahaan.

  • Aplikasi sudah di sesuikan dengan media query max-width:992px (layar smartphone).
  • File Header dan Menu di dalam nya terdapat tag <'nav'>,<'img'>, <'ul'> & <'li'>.
  • Halaman Profile berisikan profile perusahaan di dalam nya terdapat input Layout CSS,<'img'>.
  • Halaman Login berisikan <'form'> dan sudah di Validasi dengan JavaScript fungsi If Else tertera peringatan jika kolom belum terisi.
  • Halaman Transaksi berisi kan halaman untuk menginput data yang di sesuaikan dengan tag <'input'>,<'select'> dan Submit <'button'> serta sudah di beri kan validasi peringatan bila ada kolom yang belum terisi.
  • Halaman Data Transaksi berisikan <'table'> yang di dalam nya berisikan data hasil input Halaman transaksi yang sudah di sesuaikan oleh Javascript.
